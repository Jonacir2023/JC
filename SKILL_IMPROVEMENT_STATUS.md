## ✨ Aplicações do Framework

Cada skill melhorado terá:

1. **✅ Seção DO/DO NOT**
   ```
   ✅ DO: [3-5 use cases]
   ❌ DO NOT: [3-5 anti-patterns]
   ```

2. **✅ Triggers Claros**
   ```
   Ativa quando: [keywords e phrases]
   Não ativa quando: [anti-triggers]
   ```

3. **✅ Stacking Options**
   ```
   | Skill | Resultado | Uso |
   |-------|-----------|-----|
   | skill-a | descrição | contexto |
   ```

4. **✅ Platform Compatibility**
   ```
   | Platform | Suporte | Notas |
   |----------|---------|-------|
   | Opus 4.7 | ✅ Full | ... |
   ```

5. **✅ System Prompt Otimizado**
