# 📋 Project Overview

## O que é JC?

**JC** — "Dados do Claude" — é um repositório dedicado a armazenar conhecimento, documentação e contexto estruturado para trabalhos com Claude.

## Status Atual

- 🟡 **Fase**: Inicialização de Vault
- 🎯 **Objetivo**: Estabelecer estrutura de conhecimento para melhor colaboração entre Claude e equipe
- 🔄 **Stack**: Claude Code + Obsidian

## Objetivos Principais

1. Centralizar toda documentação e decisões do projeto
2. Manter contexto estruturado e reutilizável
3. Facilitar onboarding e referência rápida
4. Criar história de decisões (ADRs)
