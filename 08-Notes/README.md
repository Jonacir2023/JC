# 📝 Notas Temporárias

Espaço para notas diárias, brainstorming e ideias em progresso. **Não é conhecimento permanente** — revise regularmente e mova para folders apropriadas.

## 📅 Daily Notes

Crie notas diárias com o padrão: `YYYY-MM-DD.md`

```markdown
# 2026-06-06

## Hoje
- [ ] Tarefa 1
- [ ] Tarefa 2

## Notas
Ideias, observações, brainstorm...

## Aprendizados
Coisas importantes que descobri.
```

## 🔄 Limpeza Semanal

- ✅ Revise notas da semana
- ✅ Promova insights para pastas apropriadas
- ✅ Delete notas desatualizada
- ✅ Archive completadas

## Padrão de Arquivo

```
08-Notes/
├── 2026-06-06.md
├── 2026-06-05.md
├── Ideias/
│   ├── feature-x.md
│   └── optimization-y.md
└── Archive/
    └── 2026-05-15.md
```

---

**Regra de Ouro**: Se está em Notes por mais de 2 semanas, move para o lugar apropriado ou deleta.
