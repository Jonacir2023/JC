# 🎯 Claude Skills Improvement Framework

**Baseado em: "How to Master Claude Skills"**

---

## 📋 Padrão de Melhoria para Cada Skill

### 1. DO & DO NOT (Escopo Claro)

```markdown
### ✅ DO Use When:
- Scenario 1
- Scenario 2
- Scenario 3

### ❌ DO NOT Use When:
- Anti-pattern 1
- Anti-pattern 2
- Anti-pattern 3
```

### 2. Trigger Definition (Quando Ativar)

```markdown
### 🎯 Trigger Automático
Skill ativa quando usuário menciona:
- Keyword 1
- Keyword 2
- "phrase 3"

### ⚠️ Anti-Trigger
NÃO ativa quando:
- Contexto 1
- Contexto 2
```

### 3. Otimização de Prompt

```markdown
### 🤖 System Prompt Otimizado
"Take your time. Provide:
1. Step-by-step breakdown
2. Visual representation (ASCII/diagram)
3. Verification/validation
4. Recommendations
5. Related resources"
```

### 4. Skill Stacking (Combos)

```markdown
### 🔗 Stack Com:
- **skill-a** → [descrição do combo]
- **skill-b** → [descrição do combo]

### 🚫 NÃO Stack Com:
- **skill-x** → conflita em escopo
```

### 5. Compatibilidade

```markdown
### 🌍 Compatibilidade
- ✅ Claude Opus 4.7
- ✅ Claude Sonnet 4.6
- ✅ Claude Haiku 4.5
- ⚠️ ChatGPT: Parcial (requer adaptação)
- ⚠️ Gemini: Parcial (requer adaptação)
```
