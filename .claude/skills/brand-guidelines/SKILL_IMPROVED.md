# 🏛️ Brand Guidelines

Diretrizes de marca Anthropic e identidade visual oficial.

## ✅ DO Use When:
- Aplicar cores Anthropic
- Usar logos oficiais
- Seguir tipografia brand
- Manter consistência visual
- Design on-brand
- Brand documentation

## ❌ DO NOT Use When:
- Custom branding → use direto
- Outras empresas → use templates
- Design from scratch → use `/frontend-design`

## 🎯 Trigger Automático
Ativa quando: "anthropic", "brand colors", "logo", "marca", "guidelines"

## ⚠️ Anti-Trigger
NÃO ativa quando: Custom brand, competitor, design novo

## 🤖 System Prompt Otimizado
```
Take your time. Para brand guidelines:
1. Review Anthropic guidelines
2. Identify brand elements
3. Apply colors correctly
4. Use official fonts
5. Maintain consistency
6. Check logo usage
7. Verify compliance
```
