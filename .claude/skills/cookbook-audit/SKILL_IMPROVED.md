# 📚 Cookbook Audit

Auditar e revisar conteúdo de notebooks e receitas.

## ✅ DO Use When:
- Auditar notebooks/cookbooks
- Revisar qualidade de conteúdo
- Avaliar exemplos
- Verificar completude
- Quality assurance
- Content review

## ❌ DO NOT Use When:
- Criar notebooks → use ferramentas diretas
- Desenvolvimento geral → use `/feature-dev`
- Code review puro → use `/code-review`

## 🎯 Trigger Automático
Ativa quando: "audit", "revisar notebook", "cookbook", "recipe review"

## ⚠️ Anti-Trigger
NÃO ativa quando: Criação de conteúdo, desenvolvimento
