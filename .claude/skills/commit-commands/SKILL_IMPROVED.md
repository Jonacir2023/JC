# 📤 Commit Commands

Operações git, commits estruturados e branching.

## ✅ DO Use When:
- Executar operações git
- Criar commits estruturados
- Gerenciar branches
- Fazer versioning
- Push/pull automático
- Git workflow

## ❌ DO NOT Use When:
- Modificar código → use desenvolvimento direto
- Review de código → use `/code-review`
- Planejamento → use `/project-management`

## 🎯 Trigger Automático
Ativa quando: "commit", "git", "push", "branch", "pull", "merge"

## ⚠️ Anti-Trigger
NÃO ativa quando: Modificação de código, review, planejamento
