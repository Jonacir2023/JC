# Iniciar Projeto

Use Claude para gerar automaticamente o escopo, riscos e stakeholders do projeto.

## Comando

```
/project-management iniciar projeto:
<descrição do projeto>
```

## O que Claude faz automaticamente

1. **Geração de Escopo Macro**
   - Define o escopo geral
   - Identifica deliverables principais
   - Define objetivos SMART

2. **Brainstorming de Riscos**
   - Identifica riscos potenciais
   - Classifica por probabilidade/impacto
   - Sugere planos de mitigação

3. **Análise de Stakeholders**
   - Identifica stakeholders
   - Classifica por interesse/poder
   - Define estratégia de comunicação

4. **Documentação Inicial**
   - Charter do projeto
   - Termo de abertura
   - Matriz de comunicação

## Exemplo

```
/project-management iniciar projeto:
Implementar sistema ERP para empresa de 200 funcionários
Duração estimada: 6 meses
Orçamento: R$ 500.000
Departamentos envolvidos: TI, Financeiro, RH, Vendas
```

Retorna automaticamente:
- Escopo completo
- Lista de riscos priorizada
- Matriz de stakeholders
- Plano de comunicação
- Charter pronto para assinar
