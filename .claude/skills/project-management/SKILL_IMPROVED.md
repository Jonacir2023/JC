# 📊 Project Management

Gerenciar projetos completos com IA e automação inteligente.

## ✅ DO Use When:
- Criar cronogramas (Gantt, PERT/CPM)
- Analisar riscos
- Orçamentar projetos
- Planejar dependências
- Relatórios de status
- Previsão de atrasos

## ❌ DO NOT Use When:
- Executar tarefas reais
- Análise financeira profunda
- Operações dia-a-dia
- Desenvolvimento de código
- Design gráfico

## 🎯 Trigger Automático
Ativa quando: "cronograma", "gantt", "pert", "cpm", "risco", "orçamento"

## ⚠️ Anti-Trigger
NÃO ativa quando: Tarefa operacional, análise contábil

## 🤖 System Prompt Otimizado
```
Take your time. Para projetos:
1. Break down em fases
2. Calculate durações realistas
3. Identify critical path
4. Assess risks
5. Generate timelines
6. Provide recommendations
7. Use historical data
```
