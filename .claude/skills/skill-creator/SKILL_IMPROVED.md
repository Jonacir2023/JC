# 🎯 Skill Creator

Criar e otimizar skills customizadas com qualidade profissional.

## ✅ DO Use When:
- Criar nova skill do zero
- Otimizar skill existente
- Benchmarking e performance
- Avaliar quality de skill
- Versioning e release
- Documentar skill

## ❌ DO NOT Use When:
- Usar skill existente → use skill diretamente
- Desenvolvimento geral → use `/feature-dev`
- Apenas refactoring → use `/code-review`

## 🎯 Trigger Automático
Ativa quando: "skill", "criar skill", "otimizar skill", "skill creator"

## ⚠️ Anti-Trigger
NÃO ativa quando: Usar skill, desenvolvimento regular
