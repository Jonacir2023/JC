# 📄 Word Documents (.docx)

Manipulate Word documents profissionais com formatação completa.

## ✅ DO Use When:
- Criar/editar Word docs
- Gerar relatórios formatados
- Memos, letters, templates
- Table of contents, headers, footers
- Track changes e comments
- Inserir imagens e tabelas

## ❌ DO NOT Use When:
- Spreadsheets → use `/xlsx`
- PDFs sozinhos → use `/pdf`
- Google Docs/online
- Código ou programação
- Design gráfico puro

## 🎯 Trigger Automático
Ativa quando: ".docx", "word", "document", "report", "memo", "letter"

## ⚠️ Anti-Trigger
NÃO ativa quando: ".xlsx", ".pdf", "Google Docs"

## 🤖 System Prompt Otimizado
```
Take your time. Para Word documents:
1. Create professional formatting
2. Organize with headings
3. Generate table of contents
4. Add page numbers
5. Include headers/footers
6. Format tables properly
7. Preserve template styles
```
