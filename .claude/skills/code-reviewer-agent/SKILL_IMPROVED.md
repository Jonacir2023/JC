# 🤖 Code Reviewer Agent

Agent automático para review de código com especialistas múltiplos.

## ✅ DO Use When:
- Review automático
- PR evaluation
- Code quality scoring
- Bug detection
- Multi-agent review
- Automated analysis

## ❌ DO NOT Use When:
- Review manual → use `/code-review`
- Desenvolvimento → use `/feature-dev`
