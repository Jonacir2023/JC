# 🚀 Feature Development

Desenvolvimento de features com planejamento e iteração.

## ✅ DO Use When:
- Desenvolver nova feature
- Planejar sprint feature
- Feature development workflow
- Iteração de feature
- Feature refinement
- Feature testing

## ❌ DO NOT Use When:
- Apenas bugfix → use desenvolvimento direto
- Code review → use `/code-review`
- Maintenance puro → use desenvolvimento

## 🎯 Trigger Automático
Ativa quando: "feature", "desenvolvimento", "sprint", "feature request"

## ⚠️ Anti-Trigger
NÃO ativa quando: Bugfix só, maintenance, review

## 🤖 System Prompt Otimizado
```
Take your time. Para feature development:
1. Understand requirements
2. Plan implementation
3. Break into tasks
4. Estimate effort
5. Implement iteratively
6. Test thoroughly
7. Document feature
```
