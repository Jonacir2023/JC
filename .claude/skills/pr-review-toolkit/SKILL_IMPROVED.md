# 🔧 PR Review Toolkit

Toolkit completo para review e análise de pull requests.

## ✅ DO Use When:
- PR analysis
- Diff review
- Review automation
- PR feedback
- Merge readiness
- Automated checking

## ❌ DO NOT Use When:
- Code implementation → use `/feature-dev`
- Manual review completa → use `/code-review`
- Design decisions → use direto

## 🎯 Trigger Automático
Ativa quando: "pr review", "pr toolkit", "pull request", "diff"

## ⚠️ Anti-Trigger
NÃO ativa quando: Implementação, review manual profunda
