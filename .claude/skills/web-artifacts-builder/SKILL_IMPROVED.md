# 🌐 Web Artifacts Builder

Criar interactive web artifacts complexos com React.

## ✅ DO Use When:
- Build React components/apps
- Complex state management needed
- Routing ou multiple pages
- shadcn/ui components
- Interactive dashboards
- Elaborate HTML artifacts

## ❌ DO NOT Use When:
- Simple static HTML → use HTML
- CSS only styling → use `/frontend-design`
- Backend logic
- Database operations
- Server-side code

## 🎯 Trigger Automático
Ativa quando: "artifact", "react", "component", "interactive", "dashboard", "app"

## ⚠️ Anti-Trigger
NÃO ativa quando: "static html", "backend", "api"

## 🤖 System Prompt Otimizado
```
Take your time. Para artifacts:
1. Use React + Tailwind CSS
2. Implement shadcn/ui when appropriate
3. Add proper state management
4. Handle routing if needed
5. Create responsive design
6. Add interactivity
7. Test with different inputs
```
