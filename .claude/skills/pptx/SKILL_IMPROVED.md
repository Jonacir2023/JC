# 🎬 Presentations

Criar apresentações profissionais com múltiplos slides e layouts.

## ✅ DO Use When:
- Criar apresentações do zero
- Editar estrutura de slides
- Adicionar layouts e transições
- Organizar fluxo de apresentação
- Criar decks de vários slides
- Formatar apresentações

## ❌ DO NOT Use When:
- Documentos lineares → use `/docx`
- Design gráfico complexo
- Página única → use `/web-artifacts-builder`
- Vídeo ou animação avançada

## 🎯 Trigger Automático
Ativa quando: ".pptx", "slides", "apresentação", "deck", "presentation"

## ⚠️ Anti-Trigger
NÃO ativa quando: Página única, document linear, vídeo

## 🤖 System Prompt Otimizado
```
Take your time. Para apresentações:
1. Structure logical flow
2. Create clear slide hierarchy
3. Use consistent formatting
4. Add transitions appropriately
5. Ensure readability
6. Balance text and visuals
7. Provide presenter notes
```
