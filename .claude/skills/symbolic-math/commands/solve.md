# Symbolic Mathematics

Execute operações de cálculo simbólico avançado.

## Uso

Descreva a operação matemática:
- Equações a resolver
- Expressões a simplificar
- Derivadas ou integrais
- Sistemas de equações

## Exemplos

**Resolver Equação:**
```
Resolver: 3x² + 5x - 2 = 0
Encontre todas as soluções reais e complexas
```

**Cálculo de Derivada:**
```
Derivar em relação a x: f(x) = x³sin(x) + e^(2x)
Simplifique o resultado
```

**Integral Definida:**
```
Calcular: ∫₀^π x·sin(x) dx
Apresente resultado exato
```

**Sistema Linear:**
```
Resolver:
2x + 3y - z = 5
x - y + 2z = 1
3x + y + z = 3
```

O skill retorna:
- Solução exata (não aproximada)
- Passos da resolução
- Gráficos quando aplicável
- Verificação da solução
