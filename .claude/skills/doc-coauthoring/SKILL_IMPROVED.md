# 📝 Document Coauthoring

Colaboração em documentos, specs e proposals.

## ✅ DO Use When:
- Escrever design docs
- Colaborar em proposals
- Criar especificações
- Coordenar documentação
- Estruturar documentos complexos
- Documentação colaborativa

## ❌ DO NOT Use When:
- Código → use desenvolvimento direto
- Chat em tempo real → use Slack
- Documento simples → use `/docx`

## 🎯 Trigger Automático
Ativa quando: "spec", "proposal", "design doc", "colaborar", "coautor"

## ⚠️ Anti-Trigger
NÃO ativa quando: Código, chat, documento simples
