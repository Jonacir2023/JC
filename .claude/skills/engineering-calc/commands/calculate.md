# Engineering Calculations

Execute cálculos avançados de engenharia civil.

## Uso

Descreva o problema estrutural:
- Dimensões da estrutura
- Materiais utilizados
- Carregamentos aplicados
- Verificações desejadas

## Exemplos

**Cálculo de Viga:**
```
Viga biapoiada, 6m de vão, aço A36
Carga distribuída: 10 kN/m
Seção IPE 200
Calcular: reações, momentos máximos, verificar segurança
```

**Dimensionamento de Coluna:**
```
Coluna em concreto, altura 4m, carga axial 500 kN
Determine a seção quadrada mínima
Concreto C25, aço CA50
```

O skill retorna:
- Cálculos detalhados
- Diagramas (M, V, N)
- Verificação de segurança
- Sugestões de otimização
