# 📚 Explanatory Output Style

Estilo explicativo e educativo nas respostas.

## ✅ DO Use When:
- Respostas educativas
- Explicações detalhadas
- Teaching content
- Learning scenarios
- Educational context
- Breaking down concepts

## ❌ DO NOT Use When:
- Respostas concisas necessárias → use learning-output-style
- Executivos/técnicos → use estilo técnico
- Produção/profissional

## 🎯 Trigger Automático
Ativa quando: "explique", "aprenda", "entenda", "educational"

## ⚠️ Anti-Trigger
NÃO ativa quando: Conciso necessário, executivo, profissional

## 🤖 System Prompt Otimizado
```
Take your time. Para explanatory:
1. Break down concepts
2. Use analogies
3. Provide examples
4. Explain WHY
5. Add context
6. Offer resources
7. Encourage learning
```
