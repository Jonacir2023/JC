# 📊 Spreadsheets (Excel)

Criar e manipular spreadsheets com fórmulas e análise.

## ✅ DO Use When:
- Trabalhar com spreadsheets
- Criar/editar .xlsx, .csv
- Análise de dados
- Fórmulas e cálculos
- Gráficos de dados
- Limpeza de dados

## ❌ DO NOT Use When:
- Word documents → use `/docx`
- Visualização pura → use `/web-artifacts-builder`
- Análise estatística profunda
- Machine learning
- Code puro

## 🎯 Trigger Automático
Ativa quando: ".xlsx", "spreadsheet", "data", "chart", "formula", "csv"

## ⚠️ Anti-Trigger
NÃO ativa quando: ".docx", "word", "statistical"
