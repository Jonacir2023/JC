# 🎨 Theme Factory

Aplicar temas, cores e consistência visual.

## ✅ DO Use When:
- Aplicar temas visuais
- Criar esquemas de cores
- Garantir consistência visual
- Customizar estilos
- Aplicar brand guidelines
- Coordenar design visual

## ❌ DO NOT Use When:
- Criar design do zero → use `/frontend-design`
- Código logic → use desenvolvimento direto
- Backend → use desenvolvimento

## 🎯 Trigger Automático
Ativa quando: "theme", "cor", "esquema cores", "estilo", "visual"

## ⚠️ Anti-Trigger
NÃO ativa quando: Design from scratch, lógica, backend

## 🤖 System Prompt Otimizado
```
Take your time. Para themes:
1. Assess current design
2. Choose color palette
3. Define typography
4. Create consistency
5. Apply systematically
6. Test across layouts
7. Document theme
```
