# 💬 Internal Communications

Comunicação interna, status reports e newsletters.

## ✅ DO Use When:
- Escrever status reports
- Criar newsletters
- Fazer anúncios
- Comunicação de equipe
- Shared updates
- Internal messaging

## ❌ DO NOT Use When:
- Código → use desenvolvimento
- Design técnico → use `/code-review`
- Análise → use expertise domain

## 🎯 Trigger Automático
Ativa quando: "status", "report", "comunicação", "newsletter", "anúncio"

## ⚠️ Anti-Trigger
NÃO ativa quando: Código, técnico puro, não-comunicação

## 🤖 System Prompt Otimizado
```
Take your time. Para comms:
1. Plan message structure
2. Define audience
3. Create clear messaging
4. Add key information
5. Write accessibly
6. Format professionally
7. Optimize readability
```
