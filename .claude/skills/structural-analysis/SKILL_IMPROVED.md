# 🏢 Structural Analysis

Análise estrutural avançada com método dos elementos finitos.

## ✅ DO Use When:
- Análise com FEM
- Cálculo de deslocamentos
- Frequências naturais/modos
- Análise modal
- Análise dinâmica
- Análise de estabilidade

## ❌ DO NOT Use When:
- Cálculos simples → use `/engineering-calc`
- Design gráfico
- Código ou programação
- Análise financeira
- Projeto arquitetônico

## 🎯 Trigger Automático
Ativa quando: "analisar estrutura", "deslocamento", "modo vibração", "FEM", "elementos finitos"

## ⚠️ Anti-Trigger
NÃO ativa quando: Cálculo simples, design estético
