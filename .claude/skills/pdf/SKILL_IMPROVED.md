# 📄 PDF Generation & Conversion

Gerar, converter e manipular PDFs com qualidade profissional.

## ✅ DO Use When:
- Gerar PDFs a partir de documentos
- Converter múltiplos formatos para PDF
- Exportar relatórios estruturados
- Criar documentos com layout profissional
- Extrair dados de PDFs
- Combinar múltiplos arquivos

## ❌ DO NOT Use When:
- Editar imagens → use `/canvas-design`
- Trabalhar com vídeo
- Código ou lógica de programa
- Design gráfico complexo

## 🎯 Trigger Automático
Ativa quando: ".pdf", "pdf", "converter para pdf", "exportar pdf", "gerar pdf"

## ⚠️ Anti-Trigger
NÃO ativa quando: Edição de imagem, vídeo, ou lógica de código
