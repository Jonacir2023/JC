# 🧪 Web App Testing

Testes de web app, UI verification e automação.

## ✅ DO Use When:
- Testar UI/UX interativa
- Criar casos de teste
- Verificar com screenshots
- Automação de testes web
- Testing em navegador
- Verificar responsividade

## ❌ DO NOT Use When:
- Testes de backend/unitários → use ferramentas diretas
- Performance testing → use ferramentas especializadas
- Code implementation → use `/feature-dev`

## 🎯 Trigger Automático
Ativa quando: "test", "ui", "browser", "screenshot", "testing", "qa"

## ⚠️ Anti-Trigger
NÃO ativa quando: Backend testing, performance, implementação
