# 🚀 GUIA SUPER SIMPLES - Como Abrir o Infográfico

## ✅ MÉTODO 1: Abrir em Qualquer Navegador (MAIS FÁCIL)

### Passo 1: Abrir o Terminal
```
Abra o Terminal/Command Prompt
```

### Passo 2: Copie e Cole Este Comando
```bash
cd /home/user/JC && python3 simple_server.py
```

### Passo 3: Pressione Enter

Você verá:
```
✅ SERVIDOR INICIADO

🌐 Abra no navegador:
   http://localhost:8000/infographic-claude-prompts.html
```

### Passo 4: Copie Este Link
```
http://localhost:8000/infographic-claude-prompts.html
```

### Passo 5: Abra em Qualquer Navegador
- **MacBook:** Safari, Chrome, Firefox
- **iPhone:** Safari
- **Windows:** Chrome, Edge, Firefox

Cole o link na barra de endereço e pressione Enter.

✅ **PRONTO!** O infográfico abre!

## ❌ Se Não Funcionar

### Erro: "Porta já em uso"
```bash
# Tente outra porta:
cd /home/user/JC && python3 simple_server.py
# Se disser "Porta em uso", faça:
sudo lsof -i :8000
# E mate o processo
```

### Erro: "Python3 não encontrado"
```bash
# Instale Python3:
sudo apt-get install python3
```

### Erro: "Arquivo não encontrado"
```bash
# Verifique se está no diretório correto:
cd /home/user/JC
ls -la infographic-claude-prompts.html
```
