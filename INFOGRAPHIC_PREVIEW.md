## 🚀 **Como Usar Este Guia**

1. ✅ **Escolha uma categoria** → Selecione a área em que deseja trabalhar
2. ✅ **Copie o prompt** → Pegue o prompt exato e copie para Claude
3. ✅ **Substitua placeholders** → Troque `[NICHO]`, `[AUDIÊNCIA]`, etc. por seus valores
4. ✅ **Execute e refine** → Teste o resultado e faça ajustes
5. ✅ **Combine prompts** → Use múltiplos prompts para resultados completos

## 💡 **Dicas Profissionais**

### Para Melhores Resultados:
- Seja específico ao preencher os placeholders
- Combine prompts de categorias diferentes
- Ajuste o nível de detalhe conforme necessário
- Reutilize resultados como base para novos prompts

### Integração com Claude:
- Copie um prompt por vez
- Cole diretamente no chat do Claude
- Use as 3 versões conforme sua necessidade:
  - **Opus** → Máxima qualidade, análise profunda
  - **Sonnet** → Equilíbrio velocidade/qualidade
  - **Haiku** → Rápido, ideal para brainstorming
